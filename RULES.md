# Grid Kick Rules

## Pregame 

Each User has a Team of four field player an one goal keeper.
Each Field Player has a name and an Skillset containing: Speed, Shoot, Passing, Dribbling, Defense, Physics. 

Each User decides for a side of the field and place all its players in its own half. they must attack the goal on the ohter end of the field.

One user is randomly selected. This player can place the ball mark at any hex along the middleline, then the other user will start with the first round.

## Round Loop

A User has basically two actions per round. An action can be one of

- Running
- Passing 
- Discard

### Running
The user can select any field player from its own team and walk as much (free) field as his speed skill.

### Passing
The user can select a field player from its own team which is on the same field as the ball mark. He can then choose any direction an move the ball mark on a straight line for as much (free) fields as his passing skill.

### Discard
Discard the action and do nothing.
